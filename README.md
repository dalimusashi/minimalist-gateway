This site is built with Next.js and is optimized for zero-cost hosting on **Vercel** or **Firebase**.

### Step 1: Push to GitHub
If you see terminal errors like `command not found` when pasting, try typing the commands one by one:

1.  **Initialize Git**:
    ```bash
    git init
    ```
2.  **Add your files**:
    ```bash
    git add .
    ```
3.  **Commit the changes**:
    ```bash
    git commit -m "Initial site launch"
    ```
4.  **Set the branch to main**:
    ```bash
    git branch -M main
    ```
5.  **Add your remote repository**:
    ```bash
    git remote add origin https://github.com/dalimusashi/minimalist-gateway.git
    ```
6.  **Push the code**:
    ```bash
    git push -u origin main
    ```
    *(Note: You might be asked to log in to GitHub at this step.)*

### Step 2: Deploy to Vercel (Recommended)
1. Log in to [Vercel.com](https://vercel.com).
2. Click **"Add New > Project"** and import your GitHub repo `minimalist-gateway`.
3. **Important**: Under "Environment Variables", add:
   - `RESEND_API_KEY`: (Get this for free from [Resend.com](https://resend.com))
4. Click **Deploy**.

### Step 3: Connect Your Domain
1. In your Vercel project, go to **Settings > Domains**.
2. Type your domain (e.g., `gabrielduro.com`).
3. Follow the DNS instructions provided by Vercel to update your records at your domain registrar.
